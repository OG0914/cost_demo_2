# 成本对比模块

> 本文档详细说明成本对比功能，用于对比多个成本分析的差异。

---

## 一、模块概述

### 1.1 定位和作用

成本对比模块用于对比多个成本分析（2-4个）的差异，帮助用户快速识别不同成本分析之间的价格差异、成本构成差异，为决策提供数据支持。

### 1.2 功能范围

- 选择多个成本分析进行对比
- 对比维度：基本信息、原料、工序、包材、最终价格
- 差异高亮显示
- 对比报告导出

### 1.3 与其他模块的关联

| 关联模块 | 关联关系 | 说明 |
|----------|----------|------|
| 成本分析 | 数据来源 | 对比数据来源于成本分析记录 |
| 审核流程 | 数据范围 | 只能选择已提交或已审核的成本分析进行对比 |

---

## 二、对比功能定位

### 2.1 使用场景

**场景1：同型号不同客户对比**
- 对比同一产品型号针对不同客户的报价差异
- 分析客户地区、数量对价格的影响

**场景2：同客户不同时期对比**
- 对比同一客户不同时期的成本分析
- 分析成本变化趋势

**场景3：不同包装配置对比**
- 对比同一型号不同包装配置的成本差异
- 选择最优包装方案

**场景4：内销外销对比**
- 对比同一产品的内销价和外销价
- 分析不同销售类型的利润差异

### 2.2 对比限制

| 限制项 | 限制值 | 说明 |
|--------|--------|------|
| 最少对比数 | 2个 | 至少需要选择2个成本分析 |
| 最多对比数 | 4个 | 最多支持4个成本分析同时对比 |
| 可选状态 | 已提交/已审核 | 只有已提交或已审核的成本分析可选 |
| 数据时效 | 实时 | 对比时读取最新数据 |

## 三、对比维度

### 3.1 基本信息对比

**对比字段**：

| 字段 | 说明 | 差异标识 |
|------|------|----------|
| 成本分析编号 | 系统自动生成 | - |
| 状态 | 审核状态 | 不同值高亮 |
| 销售类型 | 内销/外销 | 不同值高亮 |
| 客户名称 | 客户公司名称 | 不同值高亮 |
| 客户地区 | 客户所在地区 | 不同值高亮 |
| 产品型号 | 具体型号名称 | 不同值高亮 |
| 法规标准 | 所属法规 | 不同值高亮 |
| 包装配置 | 包装配置名称 | 不同值高亮 |
| 订单数量 | 订单数量 | 不同值高亮 |
| 创建时间 | 创建时间 | - |

### 3.2 原料成本对比

**对比方式**：
- 按原料名称分组对比
- 显示每个成本分析中该原料的单价、用量、小计
- 计算原料成本差异

**对比表格结构**：

| 原料名称 | 成本分析1 | 成本分析2 | 成本分析3 | 成本分析4 | 差异 |
|----------|-----------|-----------|-----------|-----------|------|
| 原料A | 单价/用量/小计 | 单价/用量/小计 | ... | ... | 最大差值 |
| 原料B | ... | ... | ... | ... | ... |
| 原料合计 | 总价 | 总价 | 总价 | 总价 | 差异率 |

**差异计算**：
- 单项差异 = 最大值 - 最小值
- 差异率 = (最大值 - 最小值) ÷ 最小值 × 100%

### 3.3 工序成本对比

**对比方式**：
- 按工序名称分组对比
- 显示每个成本分析中该工序的单价、数量、小计

**对比表格结构**：

| 工序名称 | 成本分析1 | 成本分析2 | 成本分析3 | 成本分析4 | 差异 |
|----------|-----------|-----------|-----------|-----------|------|
| 工序A | 单价/数量/小计 | 单价/数量/小计 | ... | ... | 最大差值 |
| 工序合计 | 总价 | 总价 | 总价 | 总价 | 差异率 |

### 3.4 包材成本对比

**对比方式**：
- 按包材名称分组对比
- 显示每个成本分析中该包材的单价、用量、小计

**对比表格结构**：

| 包材名称 | 成本分析1 | 成本分析2 | 成本分析3 | 成本分析4 | 差异 |
|----------|-----------|-----------|-----------|-----------|------|
| 包材A | 单价/用量/小计 | 单价/用量/小计 | ... | ... | 最大差值 |
| 包材合计 | 总价 | 总价 | 总价 | 总价 | 差异率 |

### 3.5 最终价格对比

**对比字段**：

| 字段 | 说明 | 最优值标识 |
|------|------|------------|
| 基础成本价 | 基础成本 | 最低值绿色 |
| 管销价 | 管销后价格 | 最低值绿色 |
| 最终成本价 | 最终价格 | 最低值绿色 |
| 利润档位 | 各档位价格 | 最低值绿色 |

**差异高亮规则**：
- 最低值：绿色标记
- 最高值：红色标记
- 中间值：正常显示

## 四、差异高亮显示

### 4.1 差异类型定义

| 差异类型 | 标识颜色 | 说明 |
|----------|----------|------|
| 一致 | 正常文字 | 所有对比项值相同 |
| 新增 | 绿色 | 某成本分析独有的项目 |
| 修改 | 黄色 | 值存在差异 |
| 删除 | 红色 | 某成本分析缺少的项目 |
| 最优 | 深绿色 | 价格最低（正向指标）|
| 最差 | 深红色 | 价格最高（正向指标）|

### 4.2 高亮规则

**数值型字段**：
- 显示所有值
- 最低值绿色高亮（成本类）
- 最高值红色高亮（成本类）

**文本型字段**：
- 值相同时：正常显示
- 值不同时：黄色背景高亮

**存在性字段**：
- 都存在的项目：正常显示
- 部分存在的项目：缺失的显示"-"

### 4.3 差异汇总

**汇总卡片**：
- 显示每个成本分析的总成本
- 显示与最低成本的差异金额
- 显示与最低成本的差异百分比

**示例**：
```
┌─────────────────────────────────────────────────────────────┐
│ 成本对比概览                                                │
├──────────────┬──────────────┬──────────────┬──────────────┤
│   成本分析1   │   成本分析2   │   成本分析3   │   成本分析4   │
│   ¥15.50     │   ¥14.20     │   ¥16.80     │   ¥15.00     │
│              │   最优 ✓     │   +¥2.60    │              │
│              │   -8.4%      │   +18.3%     │              │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

## 五、对比报告导出

### 5.1 导出格式

**Excel导出**：
- 文件格式：.xlsx
- 文件名：成本对比报告_YYYYMMDD.xlsx
- 包含多个工作表

**工作表结构**：

| 工作表名称 | 内容 |
|------------|------|
| 概览 | 基本信息对比、最终价格对比 |
| 原料对比 | 原料明细对比 |
| 工序对比 | 工序明细对比 |
| 包材对比 | 包材明细对比 |

### 5.2 导出内容

**概览工作表**：
- 对比成本分析的基本信息
- 各成本分析的编号、客户、型号、数量
- 各成本分析的最终价格
- 差异汇总

**明细工作表**：
- 原料/工序/包材的详细对比
- 每个项目的单价、数量、小计
- 差异标识

### 5.3 导出按钮

**按钮位置**：
- 对比页面标题右侧
- 对比表格上方工具栏

**按钮样式**：
- 次要按钮
- 图标：Download
- 文字："导出对比报告"

## 七、交互流程

### 7.1 进入对比页面

**入口1：从列表页进入**
1. 在成本分析列表页选择2-4个记录
2. 点击"对比模式"按钮
3. 跳转到对比页面

**入口2：从详情页进入**
1. 在成本分析详情页点击"对比"按钮
2. 选择要对比的其他成本分析
3. 跳转到对比页面

### 7.2 对比页面操作

**标签页切换**：
- 点击标签页切换对比维度
- 切换时保持对比项不变

**导出报告**：
- 点击"导出对比报告"按钮
- 生成Excel文件并下载

**返回列表**：
- 点击"返回"按钮
- 返回成本分析列表页

### 7.3 打印支持

**打印样式**：
- 隐藏操作按钮
- 调整布局适应纸张
- 添加页眉页脚

**打印入口**：
- 浏览器打印（Ctrl+P）
- 页面打印按钮

## 九、业务规则

### 9.1 可选范围

- 只能选择状态为"已提交"或"已审核"的成本分析
- 草稿和已退回状态的成本分析不可选
- 可选范围受用户权限限制

### 9.2 数据一致性

- 对比时读取最新数据
- 不缓存历史数据
- 编辑中的成本分析不参与对比

### 9.3 权限控制

| 操作 | 权限要求 |
|------|----------|
| 查看对比 | cost:view |
| 导出报告 | cost:view |
