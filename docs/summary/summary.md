# 成本分析管理系统 - 功能文档总览

## 一、系统概述

成本分析管理系统是一款面向制造业企业的成本核算与报价管理工具，主要用于口罩及防护类产品的成本分析、报价生成、审核流程管理以及标准成本维护。系统支持内销和外销两种销售类型，提供完整的成本计算、审核流程、版本控制等功能。

### 1.1 系统定位

- **核心业务**：产品成本核算与报价管理
- **目标用户**：业务员、审核员、采购人员、生产人员、管理员
- **产品类型**：口罩、半面罩、全面罩等防护类产品

### 1.2 主要功能模块

| 序号 | 模块名称 | 功能定位 | 主要用户 |
|------|----------|----------|----------|
| 1 | 认证与权限 | 用户登录、权限控制、角色管理 | 所有用户 |

### 2.3 核心数据流向

1. **成本分析创建流程**
   ```
   基础数据(原料/包材/工序) → 成本分析创建 → 提交审核 → 审核通过 → 标准成本
   ```

2. **成本计算流程**
   ```
   原料总价 + 工序总价×系数 + 包材总价 + 运费 → 基础成本 → 管销价 → 最终价格
   ```

3. **审核流程**
   ```
   草稿 → 提交 → 待审核 → 审核通过/退回 → 已审核/已退回
   ```

### 5.2 业务规则速查

#### 成本计算公式

```
基础成本 = 原料总价 + (工序总价 × 工价系数) + 包材总价 + 运费
管销价 = 基础成本 ÷ (1 - 管销率)
内销价 = 管销价 × (1 + 增值税率) + 管销后原料
外销价 = 管销价 ÷ 汇率 + 管销后原料
保险价 = 外销价 × (1 + 保险率)
```

#### 审核状态流转

```
draft(草稿) → submitted(已提交) → approved(审核通过)
                              ↘ rejected(已退回) → resubmitted(重新提交)
```

## 六、术语表
