# 标准成本模块

## 一、模块概述

### 1.1 定位和作用

标准成本模块用于管理审核通过的成本分析作为标准参考价格。每个包装配置可以有一个当前标准成本版本和多个历史版本，支持版本追溯和恢复。标准成本可被业务员在创建新的成本分析时引用复制。

### 1.2 功能范围

- 标准成本列表
- 标准成本详情查看
- 历史版本管理
- 版本恢复
- 复制标准成本
- 标准成本对比
- 删除标准成本

### 1.3 与其他模块的关联

| 关联模块 | 关联关系 | 说明 |
|----------|----------|------|
| 成本分析 | 数据来源 | 审核通过的成本分析可设为标准成本 |
| 成本分析 | 数据引用 | 创建成本分析时可复制标准成本 |
| 审核流程 | 流程关联 | 只有审核通过的成本分析才能设为标准成本 |
| 基础数据 | 数据关联 | 标准成本关联包装配置和型号 |

## 二、版本控制机制

### 2.1 版本概念

- **当前版本**：每个包装配置+销售类型组合只能有一个当前版本
- **历史版本**：被新版本替换的旧版本自动归档为历史版本
- **版本号**：从1开始递增，每次创建新版本时自动+1

### 2.2 版本创建流程

```
审核通过的成本分析
        ↓
点击"设为标准成本"
        ↓
检查是否存在当前版本
        ↓
存在：将旧版本标记为历史
        ↓
创建新版本（版本号+1）
        ↓
标记为当前版本
```

### 2.3 版本恢复

- **功能**：基于历史版本创建新版本
- **结果**：历史版本内容 + 新版本号
- **用途**：回滚到之前的标准成本配置

## 三、标准成本列表

### 3.1 页面定位

标准成本列表页面展示所有当前有效的标准成本，支持搜索、筛选、查看历史版本等操作。

### 3.2 初始化逻辑

1. 加载标准成本列表数据（后端分页）
2. 加载产品类别列表
3. 加载用户权限信息
4. 根据权限显示操作按钮

### 3.3 功能详细描述

#### 3.3.1 搜索功能

- **搜索框位置**：列表上方左侧
- **搜索范围**：型号、包装配置、成本分析号
- **搜索方式**：防抖搜索（300ms延迟）

#### 3.3.2 筛选功能

**产品类别筛选**：
- 选项：全部、口罩、半面罩、全面罩等
- 交互：选择后立即触发查询

**销售类型筛选**：
- 选项：全部、内销、外销
- 交互：选择后立即触发查询

#### 3.3.3 列表展示

**表格列**：

| 列名 | 说明 | 特殊显示 |
|------|------|----------|
| 选择框 | 多选复选框 | 用于批量对比 |
| 成本分析编号 | 来源成本分析编号 | 可能为空 |
| 产品类别 | 型号所属类别 | - |
| 产品型号 | 具体型号名称 | - |
| 包装方式 | 包装配置名称 | 下方显示包装规格 |
| 订单数量 | 标准成本对应的数量 | 格式化显示 |
| 销售类型 | 内销/外销 | 类型徽章 |
| 最终成本价 | 标准成本价格 | 带货币单位 |
| 版本 | 版本号 | "V"前缀 |
| 设置人 | 设置标准成本的用户 | - |
| 设置时间 | 设置时间 | 格式化显示 |
| 操作 | 行内操作 | 查看/复制/历史/删除 |

#### 3.3.4 行内操作

| 操作 | 图标 | 权限要求 | 说明 |
|------|------|----------|------|
| 查看 | 眼睛图标 | cost:view | 查看关联的成本分析详情 |
| 复制 | 复制图标 | cost:create | 复制标准成本创建新成本分析 |
| 历史 | 时钟图标 | cost:manage | 查看历史版本 |
| 删除 | 删除图标 | cost:manage | 删除标准成本及所有历史版本 |

### 3.4 界面设计描述

#### 3.4.1 整体布局

- **顶部**：页面标题 + 对比模式按钮
- **筛选区**：搜索框 + 产品类别筛选 + 销售类型筛选
- **主体**：数据表格
- **底部**：分页组件

#### 3.4.2 对比模式

- **入口**："对比模式"按钮
- **条件**：选择2-4个标准成本
- **结果**：跳转到成本对比页面

### 3.5 按钮设计描述

| 按钮 | 样式 | 位置 | 交互 |
|------|------|------|------|
| 对比模式 | 次要按钮 | 标题右侧 | 选择不足2个时禁用 |
| 查看 | 图标按钮 | 表格行内 | 跳转成本分析详情 |
| 复制 | 图标按钮 | 表格行内 | 跳转新增成本分析页 |
| 历史 | 图标按钮 | 表格行内 | 打开历史版本弹窗 |
| 删除 | 图标按钮（红色） | 表格行内 | 确认后删除 |

## 四、历史版本管理

### 4.1 功能定位

历史版本管理用于查看和恢复标准成本的历史版本，每个标准成本可以有多个历史版本。

### 4.2 历史版本弹窗

#### 4.2.1 弹窗结构

- **标题**："标准成本历史版本"
- **主体**：历史版本列表表格
- **底部**：关闭按钮

#### 4.2.2 列表展示

**表格列**：

| 列名 | 说明 | 特殊显示 |
|------|------|----------|
| 版本 | 版本号 | "V"前缀 |
| 数量 | 对应的订单数量 | 格式化显示 |
| 最终价格 | 该版本的最终价格 | 带货币单位 |
| 设置人 | 设置该版本的用户 | - |
| 设置时间 | 设置时间 | 格式化显示 |
| 状态 | 当前/历史 | 状态徽章 |
| 操作 | 恢复按钮 | 历史版本显示恢复按钮 |

#### 4.2.3 版本恢复

- **入口**：历史版本行的"恢复"按钮
- **确认**：二次确认对话框
- **说明**："确定要基于 V{x} 的内容创建新版本吗？将生成 V{new} 作为当前版本。"
- **结果**：创建新版本，原当前版本变为历史版本

### 4.3 界面设计描述

#### 4.3.1 弹窗布局

- **宽度**：900px
- **高度**：自适应，最大70vh
- **表格**：可滚动

## 五、复制标准成本

### 5.1 功能定位

复制标准成本功能允许业务员基于已有的标准成本快速创建新的成本分析，减少重复录入工作。

### 5.2 操作流程

1. 在标准成本列表点击"复制"按钮
2. 系统跳转到新增成本分析页面
3. 自动填充标准成本的数据：
   - 型号配置
   - 成本明细（原料、工序、包材）
   - 数量
   - 销售类型
4. 用户修改客户信息和其他需要调整的内容
5. 提交新的成本分析

### 5.3 数据填充规则

- **保留字段**：型号配置、成本明细、数量、销售类型
- **清空字段**：客户信息、编号、状态
- **重新计算**：根据当前系统配置重新计算价格

## 六、业务规则

### 6.1 标准成本创建规则

| 条件 | 说明 |
|------|------|
| 来源 | 必须来自审核通过的成本分析 |
| 权限 | 仅管理员和审核员可设置 |
| 关联 | 必须有关联的包装配置 |
| 唯一性 | 每个包装配置+销售类型只能有一个当前版本 |

### 6.2 版本管理规则

| 规则 | 说明 |
|------|------|
| 版本递增 | 新版本号 = 最大版本号 + 1 |
| 当前版本唯一 | 设置新版本时，旧版本自动变为历史 |
| 历史版本只读 | 历史版本不能直接修改 |
| 恢复创建新版本 | 恢复历史版本时创建的是新版本，不是直接替换 |

### 6.3 删除规则

| 规则 | 说明 |
|------|------|
| 权限 | 仅管理员可删除 |
| 范围 | 删除标准成本会删除所有历史版本 |
| 引用检查 | 被引用的标准成本不能删除 |
| 确认 | 需要二次确认 |

### 6.4 引用保护

- **机制**：标准成本被成本分析引用时不能删除
- **提示**："该报价单已被业务员复制引用，无法删除"
- **检查时机**：删除操作前检查
