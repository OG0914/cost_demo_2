# 成本计算规则说明

> 本文档详细说明成本分析管理系统的计算规则，用于指导开发人员实现成本计算功能。

---

## 一、计算规则概述

成本计算是系统的核心功能，涉及原料、工序、包材、运费等多个维度的计算。系统根据产品型号分类（口罩类/半面罩类）和计算类型采用不同的计算公式。

### 1.1 计算流程概览

```
原料计算 → 工序计算 → 包材计算 → 基础成本 → 管销价 → 最终价格 → 利润区间
```

### 1.2 关键参数

| 参数名称 | 默认值 | 配置位置 | 说明 |
|---------|--------|---------|------|
| 口罩类系数 | 0.97 | 系统配置 | 口罩类原料计算系数 |
| 半面罩类系数 | 0.99 | 系统配置 | 半面罩类原料计算系数 |
| 工价系数 | 1.56 | 系统配置 | 工序成本计算系数 |
| 管销率 | 20% | 系统配置 | 管理销售费用比例 |
| 增值税率 | 13% | 系统配置 | 内销增值税率 |
| 汇率 | 7.2 | 系统配置 | 人民币兑美元汇率 |
| 保险率 | 0.3% | 系统配置 | 外销保险费率 |

## 二、原料计算规则矩阵

### 2.1 型号分类与计算类型

| 型号分类 | 计算类型 | 适用公式 | 系数 |
|---------|---------|---------|------|
| 口罩类 | 主体 | multiply | 0.97 |
| 口罩类 | 配件 | multiply | 0.97 |
| 半面罩类 | 主体 | multiply | 0.99 |
| 半面罩类 | 配件 | multiply | 0.99 |
| 半面罩类 | 滤毒盒 | divide | 0.99 |
| 半面罩类 | 滤棉 | divide | 0.99 |
| 半面罩类 | 滤饼 | divide | 0.99 |

### 2.2 计算公式详解

#### multiply公式

**适用场景**：口罩类所有计算类型、半面罩类主体和配件

**公式**：
```
原料小计 = (用量 × 单价) ÷ 系数
```

**参数说明**：
- 用量：每个产品使用的原料数量
- 单价：原料的单价（元/单位）
- 系数：型号分类对应的系数（口罩0.97，半面罩0.99）

**计算逻辑**：
1. 先计算原料总成本（用量 × 单价）
2. 再除以系数得到单个产品的原料成本
3. 系数用于覆盖损耗、不良品等因素

#### divide公式

**适用场景**：半面罩类滤毒盒、滤棉、滤饼

**公式**：
```
原料小计 = (单价 ÷ 用量) ÷ 系数
```

**参数说明**：
- 单价：原料的单价（元/单位）
- 用量：每箱产品使用的原料数量
- 系数：型号分类对应的系数（半面罩0.99）

**计算逻辑**：
1. 先计算单个产品的原料成本（单价 ÷ 用量）
2. 再除以系数得到最终原料成本
3. 适用于按箱采购、按个使用的原料

### 2.3 原料总价计算

```
原料总价 = Σ(各原料小计)
```

**说明**：
- 汇总所有原料明细的小计金额
- 区分管销前原料和管销后原料
- 管销后原料在最终价格计算时单独处理

---

## 三、工序计算规则

### 3.1 工序小计计算

**公式**：
```
工序小计 = 工序单价 × 数量 × 工价系数
```

**参数说明**：
- 工序单价：每道工序的单价（元/单位）
- 数量：该工序的执行次数
- 工价系数：覆盖人工和管理成本的系数（默认1.56）

### 3.2 工序总价计算

```
工序总价 = Σ(各工序小计)
```

### 3.3 工价系数说明

工价系数用于覆盖以下成本：
- 直接人工成本
- 间接人工成本
- 管理费用分摊
- 设备折旧分摊

**默认值**：1.56
**配置位置**：系统配置
**可调整性**：支持按成本分析单独调整

---

## 四、包材计算规则

### 4.1 包材小计计算

**公式**：
```
包材小计 = (单价 ÷ 用量) ÷ 系数
```

**参数说明**：
- 单价：包材的单价（元/单位）
- 用量：每箱产品使用的包材数量
- 系数：与原料系数相同（口罩0.97，半面罩0.99）

### 4.2 包材总价计算

```
包材总价 = Σ(各包材小计)
```

### 4.3 包材计算特点

- 采用divide公式计算
- 系数与原料系数保持一致
- 用量通常按箱计算

---

## 五、基础成本计算

### 5.1 基础成本公式

```
基础成本 = 原料总价 + 工序总价 + 包材总价 + 运费成本
```

### 5.2 运费成本计算

#### 运费总价计算（外销）

```
运费总价 = CBM × 每CBM单价 × 箱数
```

**参数说明**：
- CBM：外箱体积（立方米）
- 每CBM单价：运输每立方米的费用
- 箱数：订单总箱数

#### 运费成本计算

```
运费成本 = 运费总价 ÷ 订单数量
```

### 5.3 运费计入逻辑

| 销售类型 | 条款 | 运费计入基础成本 | 说明 |
|---------|------|-----------------|------|
| 内销 | - | 是 | 运费作为成本的一部分 |
| 外销 | FOB | 否 | 运费由买方承担 |
| 外销 | CIF | 是 | 运费由卖方承担 |

---

## 六、管销价计算

### 6.1 管销价公式

```
管销价 = (基础成本 + 运费) ÷ (1 - 管销率)
```

### 6.2 参数说明

- 基础成本：原料+工序+包材+运费
- 运费：根据条款决定是否计入
- 管销率：管理销售费用比例（默认20%）

### 6.3 计算示例

**示例**：
- 基础成本：10元
- 运费：0.5元
- 管销率：20%

**计算**：
```
管销价 = (10 + 0.5) ÷ (1 - 0.2) = 10.5 ÷ 0.8 = 13.125元
```

---

## 七、最终价格计算

### 7.1 内销价计算

**公式**：
```
内销价 = 管销价 × (1 + 增值税率) + 管销后原料
```

**参数说明**：
- 管销价：经过管销率调整后的价格
- 增值税率：默认13%
- 管销后原料：需要在管销后计算的特殊原料成本

**计算示例**：
- 管销价：13.125元
- 增值税率：13%
- 管销后原料：2元

**计算**：
```
内销价 = 13.125 × 1.13 + 2 = 14.831 + 2 = 16.831元
```

### 7.2 外销价计算

**公式**：
```
外销价 = 管销价 ÷ 汇率 + 管销后原料
保险价 = 外销价 × (1 + 保险率)
```

**参数说明**：
- 管销价：经过管销率调整后的价格
- 汇率：人民币兑美元汇率（默认7.2）
- 保险率：默认0.3%
- 管销后原料：需要在管销后计算的特殊原料成本

**计算示例**：
- 管销价：13.125元
- 汇率：7.2
- 保险率：0.3%
- 管销后原料：2元

**计算**：
```
外销价 = 13.125 ÷ 7.2 + 2 = 1.823 + 2 = 3.823元
保险价 = 3.823 × 1.003 = 3.834元
```

---

## 八、自定义费用计算

### 8.1 费用项结构

每个自定义费用包含：
- 费用名称：如"特殊税费"、"附加费"
- 费率：百分比（如5%表示0.05）

### 8.2 累乘计算方式

```
最终价格 = 管销价 × (1 + 费率1) × (1 + 费率2) × ... + 管销后原料
```

**计算示例**：
- 管销价：13.125元
- 费用1：5%
- 费用2：3%
- 管销后原料：2元

**计算**：
```
最终价格 = 13.125 × 1.05 × 1.03 + 2 = 14.199 + 2 = 16.199元
```

## 九、利润区间计算

### 9.1 默认档位

| 档位 | 利润率 | 用途 |
|------|--------|------|
| 档位1 | 5% | 薄利多销 |
| 档位2 | 10% | 正常利润 |
| 档位3 | 25% | 较高利润 |
| 档位4 | 50% | 高利润 |

### 9.2 计算公式

```
利润价格 = 基础价 ÷ (1 - 利润率)
```

**基础价定义**：
- 内销：管销价
- 外销：管销价 ÷ 汇率

### 9.3 计算示例

**内销示例**：
- 管销价：13.125元
- 利润率：10%

**计算**：
```
利润价格 = 13.125 ÷ (1 - 0.1) = 13.125 ÷ 0.9 = 14.583元
```

**外销示例**：
- 管销价：13.125元
- 汇率：7.2
- 利润率：25%

**计算**：
```
基础价 = 13.125 ÷ 7.2 = 1.823元
利润价格 = 1.823 ÷ (1 - 0.25) = 1.823 ÷ 0.75 = 2.431元
```

## 十、计算示例汇总

### 10.1 口罩类内销完整示例

**基础数据**：
- 原料总价：5元（multiply公式计算）
- 工序总价：3元
- 包材总价：2元
- 运费成本：0.5元
- 管销率：20%
- 增值税率：13%
- 管销后原料：1元

**计算步骤**：
```
1. 基础成本 = 5 + 3 + 2 + 0.5 = 10.5元
2. 管销价 = 10.5 ÷ (1 - 0.2) = 13.125元
3. 内销价 = 13.125 × 1.13 + 1 = 15.831元
4. 利润价格(10%) = 13.125 ÷ 0.9 = 14.583元
```

### 10.2 半面罩类外销完整示例

**基础数据**：
- 原料总价：8元（divide公式计算）
- 工序总价：4元
- 包材总价：3元
- 运费总价：500元
- 订单数量：1000个
- 管销率：20%
- 汇率：7.2
- 保险率：0.3%
- 管销后原料：1.5元

**计算步骤**：
```
1. 运费成本 = 500 ÷ 1000 = 0.5元
2. 基础成本 = 8 + 4 + 3 + 0.5 = 15.5元
3. 管销价 = 15.5 ÷ (1 - 0.2) = 19.375元
4. 外销价 = 19.375 ÷ 7.2 + 1.5 = 4.191元
5. 保险价 = 4.191 × 1.003 = 4.204元
6. 利润价格(25%) = (19.375 ÷ 7.2) ÷ 0.75 = 3.588元
```

## 十一、规则变更影响说明

### 11.1 系数变更影响

| 变更项 | 影响范围 | 是否需要重新计算 |
|--------|---------|-----------------|
| 口罩类系数 | 所有口罩类成本分析 | 是 |
| 半面罩类系数 | 所有半面罩类成本分析 | 是 |
| 工价系数 | 所有成本分析的工序成本 | 是 |
| 管销率 | 所有成本分析 | 是 |
| 增值税率 | 所有内销成本分析 | 是 |
| 汇率 | 所有外销成本分析 | 是 |
| 保险率 | 所有CIF外销成本分析 | 是 |

### 11.2 重新计算策略

**历史数据**：
- 已审核的成本分析不自动重新计算
- 保留历史计算结果用于追溯
- 新建成本分析使用最新系数

**草稿数据**：
- 打开草稿时自动使用最新系数重新计算
- 提示用户系数已更新

**标准成本**：
- 创建新版本时使用最新系数
- 历史版本保留原计算结果

### 11.3 计算精度

**金额精度**：
- 中间计算：保留6位小数
- 最终显示：保留2位小数
- 存储：保留4位小数

**百分比精度**：
- 显示：保留2位小数（如20.00%）
- 计算：使用原始值（如0.2）

## 十二、计算公式速查表

| 项目 | 公式 |
|------|------|
| 原料小计(multiply) | (用量 × 单价) ÷ 系数 |
| 原料小计(divide) | (单价 ÷ 用量) ÷ 系数 |
| 工序小计 | 单价 × 数量 × 工价系数 |
| 包材小计 | (单价 ÷ 用量) ÷ 系数 |
| 运费总价 | CBM × 每CBM单价 × 箱数 |
| 运费成本 | 运费总价 ÷ 订单数量 |
| 基础成本 | 原料总价 + 工序总价 + 包材总价 + 运费成本 |
| 管销价 | (基础成本 + 运费) ÷ (1 - 管销率) |
| 内销价 | 管销价 × (1 + 增值税率) + 管销后原料 |
| 外销价 | 管销价 ÷ 汇率 + 管销后原料 |
| 保险价 | 外销价 × (1 + 保险率) |
| 利润价格 | 基础价 ÷ (1 - 利润率) |
