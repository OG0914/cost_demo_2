# 帮助中心模块

## 一、模块概述

### 1.1 定位和作用

帮助中心模块提供系统使用指南、功能说明、操作教程等帮助文档，帮助用户快速了解和使用系统功能。支持文档搜索、目录导航、阅读进度保存等功能。

### 1.2 功能范围

- 帮助文档浏览
- 文档搜索
- 目录导航
- 阅读进度保存
- 面包屑导航

### 1.3 与其他模块的关联

| 关联模块 | 关联关系 | 说明 |
|----------|----------|------|
| 所有模块 | 入口关联 | 各模块可链接到相关帮助文档 |
| 系统配置 | 内容关联 | 系统配置说明文档 |

## 二、页面结构

### 2.1 整体布局

采用经典的左右布局设计：

```
┌─────────────────────────────────────────────────────────┐
│ 帮助中心                                    [搜索框]     │
├─────────────────────────────────────────────────────────┤
│ [面包屑导航]                                             │
├───────────────────┬─────────────────────────────────────┤
│                   │                                     │
│   侧边栏目录       │         主内容区                    │
│   (固定宽度)       │         (可滚动)                    │
│                   │                                     │
│   - 菜单项1        │         帮助文档内容                 │
│   - 菜单项2        │         (Markdown渲染)              │
│   - 菜单项3        │                                     │
│                   │                                     │
└───────────────────┴─────────────────────────────────────┘
```

### 2.2 头部区域

- **页面标题**："帮助中心"
- **面包屑导航**：当前文档路径，支持点击跳转
- **搜索框**：文档搜索功能

### 2.3 侧边栏区域

- **目录树**：帮助文档的层级目录
- **当前高亮**：当前阅读项高亮显示
- **滚动**：目录过多时可滚动

### 2.4 主内容区域

- **文档内容**：Markdown格式渲染
- **代码高亮**：代码块语法高亮
- **图片展示**：支持图片放大查看
- **内部链接**：支持文档内跳转

## 三、功能详细描述

### 3.1 文档浏览

#### 3.1.1 目录结构

帮助文档按功能模块组织：

```
帮助中心
├── 快速开始
│   ├── 系统介绍
│   ├── 登录与账号
│   └── 界面概览
├── 成本分析
│   ├── 创建成本分析
│   ├── 编辑成本分析
│   ├── 成本计算公式
│   └── 成本对比
├── 审核流程
│   ├── 提交审核
│   ├── 审核操作
│   └── 重新提交
├── 标准成本
│   ├── 设为标准成本
│   ├── 历史版本
│   └── 复制标准成本
├── 基础数据
│   ├── 原料管理
│   ├── 包材管理
│   ├── 工序管理
│   └── 型号管理
├── 系统配置
│   ├── 业务配置
│   └── 计算规则
└── 常见问题
    ├── 成本计算问题
    ├── 审核流程问题
    └── 系统使用问题
```

#### 3.1.2 文档渲染

- **格式支持**：Markdown格式
- **样式**：统一的文档样式
- **代码块**：语法高亮
- **表格**：样式美化
- **图片**：支持点击放大

### 3.2 文档搜索

#### 3.2.1 搜索框

- **位置**：头部右侧
- **占位提示**："搜索帮助文档..."
- **实时搜索**：输入时实时显示搜索结果

#### 3.2.2 搜索结果

- **展示形式**：下拉列表
- **结果项**：标题 + 内容摘要
- **高亮**：匹配关键词高亮显示
- **跳转**：点击跳转到对应文档位置

#### 3.2.3 搜索范围

- 文档标题
- 文档内容
- 代码块内容

### 3.3 目录导航

#### 3.3.1 目录树

- **层级展开**：支持多级目录展开/折叠
- **当前项高亮**：当前阅读的文档高亮显示
- **点击跳转**：点击目录项跳转到对应文档

#### 3.3.2 面包屑导航

- **位置**：头部下方
- **格式**：首页 / 一级目录 / 二级目录 / 当前文档
- **点击跳转**：点击上级目录返回

### 3.4 阅读进度

#### 3.4.1 进度保存

- **自动保存**：滚动时自动保存阅读位置
- **保存间隔**：500ms节流保存
- **保存方式**：localStorage存储

#### 3.4.2 进度恢复

- **自动恢复**：重新打开文档时自动恢复到上次位置
- **平滑滚动**：恢复时平滑滚动到目标位置

## 四、界面设计描述

### 4.1 头部设计

- **高度**：56px
- **背景**：白色
- **边框**：底部1px边框
- **布局**：flex布局，左右分布

#### 4.1.1 面包屑

- **字体**：14px
- **分隔符**："/"
- **颜色**：灰色（上级），黑色（当前）
- **悬停**：上级可点击，悬停变蓝色

#### 4.1.2 搜索框

- **宽度**：最大400px
- **样式**：圆角输入框
- **图标**：搜索图标在左侧
- **结果下拉**：白色背景，阴影边框

### 4.2 侧边栏设计

- **宽度**：240px
- **背景**：浅灰色
- **边框**：右侧1px边框
- **滚动**：独立滚动条

#### 4.2.1 菜单项

- **高度**：40px
- **字体**：14px
- **图标**：左侧16px图标
- **缩进**：每级缩进16px

#### 4.2.2 当前项样式

- **背景**：蓝色半透明
- **文字**：蓝色
- **边框**：左侧3px蓝色边框

### 4.3 主内容区设计

- **背景**：白色
- **内边距**：24px-48px（响应式）
- **最大宽度**：800px（内容区居中）
- **滚动**：独立滚动条

#### 4.3.1 文档标题

- **字体**：32px
- **字重**：bold
- **下边距**：24px

#### 4.3.2 正文样式

- **字体**：16px
- **行高**：1.8
- **段落间距**：16px

#### 4.3.3 代码块样式

- **背景**：深色背景
- **字体**：等宽字体
- **内边距**：16px
- **圆角**：8px

## 五、移动端适配

### 5.1 布局调整

- **侧边栏**：隐藏，通过菜单按钮触发抽屉
- **主内容区**：全宽显示
- **搜索框**：移至抽屉内或简化

### 5.2 抽屉菜单

- **触发**：左上角菜单按钮
- **方向**：左侧滑出
- **宽度**：80%屏幕宽度
- **遮罩**：半透明黑色遮罩

## 六、按钮设计描述

| 按钮 | 样式 | 位置 | 交互 |
|------|------|------|------|
| 菜单按钮 | 图标按钮 | 头部左侧（移动端） | 打开侧边栏抽屉 |
| 目录项 | 文字按钮 | 侧边栏 | 跳转到对应文档 |
| 面包屑项 | 文字链接 | 面包屑导航 | 跳转到上级目录 |
| 搜索结果 | 卡片按钮 | 搜索下拉 | 跳转到对应文档 |

## 七、状态与提示

### 7.1 加载状态

| 状态 | 视觉表现 |
|------|----------|
| 文档加载中 | 内容区显示骨架屏或loading |
| 搜索中 | 搜索框显示loading图标 |

### 7.2 错误状态

| 状态 | 视觉表现 | 操作 |
|------|----------|------|
| 文档加载失败 | 错误提示 + 重试按钮 | 点击重试加载 |
| 文档不存在 | 404提示 | 返回帮助首页 |

### 7.3 空状态

- **搜索无结果**："未找到相关内容，请尝试其他关键词"
- **目录为空**："暂无帮助文档"
