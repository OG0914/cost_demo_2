# 仪表盘模块

## 一、模块概述

### 1.1 定位和作用

仪表盘是用户登录系统后的默认首页，提供系统数据的概览展示、快捷功能入口和个性化导航。通过仪表盘，用户可以快速了解系统运行状态、查看待办事项、访问常用功能。

### 1.2 功能范围

- 个性化问候语展示
- 数据统计卡片
- 快捷导航配置
- 系统概况/通知展示
- 数据图表展示

### 1.3 与其他模块的关联

| 关联模块 | 关联关系 | 说明 |
|----------|----------|------|
| 成本分析 | 数据关联 | 显示成本分析统计数据 |
| 审核流程 | 数据关联 | 显示待审核数量 |
| 通知中心 | 数据关联 | 显示最近通知 |
| 所有模块 | 导航关联 | 提供快捷入口 |

## 二、页面初始化逻辑

### 2.1 数据加载流程

1. **并行加载核心数据**：
   - 统计数据（成本分析数量、待审核数量等）
   - 法规总览
   - 型号排行
   - 周报价数据
   - 待审核数量（有审核权限时）
   - 最近操作记录（非管理员）

2. **加载用户配置**：
   - 从localStorage读取用户自定义的快捷导航

3. **渲染界面**：
   - 根据用户角色显示不同的内容
   - 管理员显示系统概况
   - 非管理员显示系统通知

### 2.2 权限适配

根据用户角色和权限，仪表盘展示不同的数据：

| 用户类型 | 统计数据 | 右侧区域 |
|----------|----------|----------|
| 管理员 | 运营统计（月报价数、活跃原料等） | 系统概况 |
| 业务员/审核员 | 个人统计（我的成本分析、待审核等） | 系统通知 |
| 采购/生产 | 基础数据统计（原料数、型号数等） | 系统通知 |

## 三、功能详细描述

### 3.1 问候语区域

#### 3.1.1 功能说明

根据当前时间和用户信息，显示个性化的问候语和日期信息。

#### 3.1.2 问候语规则

| 时间段 | 问候语 |
|--------|--------|
| 06:00 - 12:00 | "早上好" |
| 12:00 - 14:00 | "中午好" |
| 14:00 - 18:00 | "下午好" |
| 18:00 - 24:00 | "晚上好" |
| 00:00 - 06:00 | "夜深了" |

#### 3.1.3 副问候语

- 工作时间（06:00-18:00）："欢迎回来，今天又是高效工作的一天~"
- 非工作时间（18:00-06:00）："系统检测到当前为下班时间段，请注意保存工作~"

#### 3.1.4 日期展示

- **日**：大号数字显示当前日期
- **月**：斜杠分隔的月份
- **星期**：中文星期几

#### 3.1.5 界面设计

- **布局**：横向卡片，左侧装饰条
- **样式**：白色背景，蓝色左侧边框，圆角设计
- **悬停效果**：边框变深，轻微阴影

### 3.2 统计卡片区域

#### 3.2.1 功能说明

以卡片形式展示关键业务数据，根据用户角色显示不同的统计项。

#### 3.2.2 管理员统计项

| 统计项 | 说明 | 图标颜色 |
|--------|------|----------|
| 本月报价数 | 当月创建的成本分析数量 | 蓝色 |
| 活跃原料数 | 系统中启用的原料数量 | 绿色 |
| 活跃型号数 | 系统中启用的型号数量 | 紫色 |
| 增长率 | 较上月增长百分比 | 橙色 |

#### 3.2.3 业务员/审核员统计项

| 统计项 | 说明 | 图标颜色 |
|--------|------|----------|
| 我的成本分析 | 当前用户创建的成本分析总数 | 蓝色 |
| 待审核 | 待审核的成本分析数量 | 橙色 |
| 已通过 | 已审核通过的成本分析数量 | 绿色 |
| 已退回 | 被退回的成本分析数量 | 红色 |

#### 3.2.4 采购/生产统计项

| 统计项 | 说明 | 图标颜色 |
|--------|------|----------|
| 活跃原料 | 系统中启用的原料数量 | 蓝色 |
| 活跃型号 | 系统中启用的型号数量 | 绿色 |
| 包材配置数 | 系统中包材配置数量 | 紫色 |
| 工序配置数 | 系统中工序配置数量 | 橙色 |

#### 3.2.5 界面设计

- **布局**：4列网格布局，响应式适配
- **卡片样式**：白色背景，顶部图标，中间数字，底部标签
- **图标**：圆形背景，白色图标
- **悬停效果**：轻微上浮，阴影加深

### 3.3 快捷导航区域

#### 3.3.1 功能说明

用户可以自定义常用功能的快捷入口，最多可添加4个快捷导航。

#### 3.3.2 默认导航

新用户或首次访问时，默认显示：
1. 新增成本分析
2. 标准成本

#### 3.3.3 可添加的导航项

| 导航项 | 图标 | 权限要求 |
|--------|------|----------|
| 新增成本分析 | 加号图标 | cost:create |
| 标准成本 | 文件列表图标 | cost:view |
| 成本记录 | 历史记录图标 | cost:view |
| 待审核 | 时间图标 | review:view |
| 已审核 | 勾选图标 | review:view |
| 原料管理 | 数据库图标 | master:material:view |
| 包材管理 | 盒子图标 | master:packaging:view |
| 工序管理 | 设置图标 | master:process:view |
| 型号管理 | 网格图标 | master:model:view |
| 法规管理 | 政府图标 | master:regulation:view |
| 客户管理 | 用户图标 | master:customer:view |

#### 3.3.4 添加导航

- **入口**：快捷导航区域的"添加"按钮（虚线边框卡片）
- **交互**：点击后弹出选择对话框
- **对话框**：网格展示所有可选导航项

#### 3.3.5 删除导航

- **方式**：鼠标悬停在快捷导航卡片上显示删除按钮
- **确认**：点击删除按钮后需要二次确认
- **提示**："确定要删除快捷方式「xxx」吗？"

#### 3.3.6 待审核角标

- **显示条件**：快捷导航包含"待审核"且有待审核数据
- **样式**：红色圆形角标，显示待审核数量
- **位置**：导航图标右上角

#### 3.3.7 界面设计

- **布局**：2行2列网格（桌面端），2列（平板），1列（手机）
- **卡片样式**：白色背景，顶部图标，底部文字
- **图标**：圆角矩形背景，彩色图标
- **添加按钮**：虚线边框，加号图标

### 3.4 系统概况区域（管理员）

#### 3.4.1 功能说明

展示系统运行状态和基本信息，仅管理员可见。

#### 3.4.2 展示内容

| 项目 | 说明 | 状态显示 |
|------|------|----------|
| 数据库状态 | 数据库连接状态 | 正常（绿色脉冲点）/异常（红色） |
| 最近备份 | 最近一次数据备份时间 | 时间文本 |
| 系统版本 | 当前系统版本号 | 版本文本 |

#### 3.4.3 界面设计

- **布局**：列表形式，每项一行
- **样式**：灰色背景行，图标+文字+状态

### 3.5 系统通知区域（非管理员）

#### 3.5.1 功能说明

展示最近的系统通知和操作记录，非管理员用户可见。

#### 3.5.2 通知类型

| 类型 | 图标颜色 | 说明 |
|------|----------|------|
| 系统通知 | 蓝色 | 一般性系统消息 |
| 待审核提醒 | 橙色 | 有待审核的成本分析 |
| 审核通过 | 绿色 | 成本分析审核通过 |
| 审核退回 | 红色 | 成本分析被退回 |
| 重要提醒 | 黄色 | 需要关注的消息 |

#### 3.5.3 通知内容

- **标题**：通知类型名称
- **内容**：通知详细说明
- **时间**：通知产生时间

#### 3.5.4 交互

- **点击跳转**：点击通知项可跳转到相关页面
  - 成本分析相关 → 成本分析详情
  - 原料相关 → 原料管理

#### 3.5.5 界面设计

- **布局**：垂直列表
- **样式**：不同背景色区分类型，圆角卡片
- **悬停效果**：轻微阴影

### 3.6 图表区域

#### 3.6.1 功能说明

以图表形式展示业务数据趋势和分布。

#### 3.6.2 周报价趋势图

- **类型**：柱状图
- **维度**：本月 vs 上月，按周对比
- **数据**：每周创建的成本分析数量

#### 3.6.3 型号排行图

- **类型**：横向柱状图
- **维度**：Top N 型号
- **数据**：各型号的成本分析数量

#### 3.6.4 界面设计

- **布局**：图表并排展示
- **样式**：白色卡片背景，圆角
- **响应式**：小屏设备堆叠显示

## 四、界面设计描述

### 4.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  问候语区域（全宽）                                       │
├─────────────────────────────────────────────────────────┤
│  统计卡片区域（4列网格）                                   │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────┐  ┌───────────────────┐ │
│  │      快捷导航区域            │  │ 系统概况/通知区域 │ │
│  │      （2/3宽度）             │  │ （1/3宽度）       │ │
│  └─────────────────────────────┘  └───────────────────┘ │
├─────────────────────────────────────────────────────────┤
│  图表区域（全宽）                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.2 响应式适配

| 屏幕尺寸 | 布局调整 |
|----------|----------|
| >= 1024px | 4列统计卡片，左右分栏 |
| 768px - 1023px | 2列统计卡片，上下堆叠 |
| < 768px | 1列布局，全部堆叠 |

## 五、按钮设计描述

| 按钮 | 样式 | 位置 | 交互 |
|------|------|------|------|
| 刷新数据 | 文字按钮（蓝色） | 页面标题右侧 | 点击刷新所有数据，显示loading |
| 添加快捷导航 | 虚线卡片 | 快捷导航网格 | 点击打开选择对话框 |
| 删除快捷导航 | 图标按钮（灰色） | 快捷导航卡片右上角 | 点击确认后删除 |
| 快捷导航项 | 卡片按钮 | 快捷导航网格 | 点击跳转到对应页面 |

## 六、状态与提示

### 6.1 页面状态

| 状态 | 视觉表现 | 说明 |
|------|----------|------|
| 初始加载 | 全屏loading或骨架屏 | 首次进入页面 |
| 数据加载 | 局部loading | 刷新数据时 |
| 正常显示 | 完整内容 | 数据加载完成 |
| 空数据 | 空状态提示 | 无数据时 |
| 错误状态 | 错误提示 | 数据加载失败 |

### 6.2 操作反馈

| 操作 | 成功提示 | 失败提示 |
|------|----------|----------|
| 刷新数据 | 无 | "加载数据失败，请重试" |
| 添加导航 | 无（直接添加） | 无 |
| 删除导航 | 无 | 无 |

## 七、数据刷新机制

### 7.1 自动刷新

- **统计数据**：页面获得焦点时自动刷新
- **待审核数量**：每5分钟自动刷新

### 7.2 手动刷新

- **刷新按钮**：点击页面标题右侧的刷新按钮
- **下拉刷新**：移动端支持下拉刷新（如实现）
