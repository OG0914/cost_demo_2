# 成本分析模块

## 一、模块概述

### 1.1 定位和作用

成本分析模块是系统的核心功能模块，用于创建、管理和查看产品成本分析记录。通过选择产品型号配置、填写客户信息、录入成本明细，系统自动计算成本价格，支持内销和外销两种销售类型。

### 1.2 功能范围

- 成本记录列表
- 新增成本分析
- 编辑成本分析
- 成本详情查看
- 成本对比
- 复制成本分析
- 删除成本分析

### 1.3 与其他模块的关联

| 关联模块 | 关联关系 | 说明 |
|----------|----------|------|
| 基础数据 | 数据依赖 | 原料、包材、工序、型号、客户、法规数据 |
| 审核流程 | 流程关联 | 成本分析提交后进入审核流程 |
| 标准成本 | 数据关联 | 审核通过可设为标准成本，可复制标准成本 |
| 系统配置 | 参数依赖 | 费率、汇率、系数等计算参数 |

### 3.4 成本计算流程

成本计算是系统的核心功能，涉及多个步骤和复杂的公式计算。以下是详细的计算流程说明。

#### 3.4.1 原料小计计算规则

原料小计根据型号分类和计算类型采用不同的计算公式。

**计算类型定义**：

| 型号分类 | 计算类型 | 说明 |
|---------|---------|------|
| 口罩类 | 主体 | 口罩主体原料 |
| 口罩类 | 配件 | 口罩配件原料 |
| 半面罩类 | 主体 | 半面罩主体 |
| 半面罩类 | 配件 | 半面罩配件 |
| 半面罩类 | 滤毒盒 | 滤毒盒原料 |
| 半面罩类 | 滤棉 | 滤棉原料 |
| 半面罩类 | 滤饼 | 滤饼原料 |

**计算公式**：

| 公式类型 | 计算公式 | 适用场景 |
|---------|---------|---------|
| multiply | 小计 = (用量 × 单价) ÷ 系数 | 口罩类主体、配件 |
| divide | 小计 = (单价 ÷ 用量) ÷ 系数 | 半面罩类滤毒盒、滤棉、滤饼 |

**系数配置**：

| 型号分类 | 默认系数 | 配置位置 |
|---------|---------|---------|
| 口罩类 | 0.97 | 系统配置 |
| 半面罩类 | 0.99 | 系统配置 |

**计算示例**：

**示例1：口罩类主体原料（multiply公式）**
- 原料单价：0.5元/片
- 用量：2片
- 系数：0.97
- 计算：(2 × 0.5) ÷ 0.97 = 1.0309元

**示例2：半面罩类滤毒盒（divide公式）**
- 原料单价：15元/个
- 用量：40个/箱
- 系数：0.99
- 计算：(15 ÷ 40) ÷ 0.99 = 0.3788元

#### 3.4.2 工价系数应用

工序成本计算需要考虑工价系数。

**计算公式**：
```
工序小计 = 工序单价 × 数量 × 工价系数
```

**工价系数**：
- 默认值：1.56
- 配置位置：系统配置
- 用途：覆盖人工成本、管理成本等间接费用

**计算示例**：
- 工序单价：0.1元/道
- 数量：3道
- 工价系数：1.56
- 计算：0.1 × 3 × 1.56 = 0.468元

**工序总价**：
```
工序总价 = Σ(各工序小计)
```

#### 3.4.3 包材计算

包材成本采用divide公式计算。

**计算公式**：
```
包材小计 = (单价 ÷ 用量) ÷ 系数
```

**系数**：
- 包材系数与原料系数相同（口罩0.97，半面罩0.99）

**计算示例**：
- 包材单价：2元/个
- 用量：50个/箱
- 系数：0.97
- 计算：(2 ÷ 50) ÷ 0.97 = 0.0412元

**包材总价**：
```
包材总价 = Σ(各包材小计)
```

#### 3.4.4 基础成本计算

基础成本是成本计算的核心中间值。

**计算公式**：
```
基础成本 = 原料总价 + (工序总价 × 工价系数) + 包材总价 + 运费成本
```

**运费成本计算**（外销）：
```
运费成本 = 运费总价 ÷ 订单数量
```

**运费是否计入基础成本的开关逻辑**：

| 场景 | 运费计入基础成本 | 说明 |
|------|-----------------|------|
| 内销 | 是 | 运费作为成本的一部分 |
| 外销-FOB | 否 | FOB条款下运费由买方承担 |
| 外销-CIF | 是 | CIF条款下运费由卖方承担 |

**运费总价计算**（外销）：
```
运费总价 = CBM × 每CBM单价 × 箱数
```

#### 3.4.5 管销价计算

管销价是基础成本加上管理销售费用的价格。

**计算公式**：
```
管销价 = (基础成本 + 运费) ÷ (1 - 管销率)
```

**管销率**：
- 默认值：20%（0.2）
- 配置位置：系统配置
- 可针对单个成本分析进行调整

**计算示例**：
- 基础成本：10元
- 运费：0.5元
- 管销率：20%
- 计算：(10 + 0.5) ÷ (1 - 0.2) = 13.125元

#### 3.4.6 最终价格计算

最终价格根据销售类型采用不同的计算公式。

**内销价计算**：
```
内销价 = 管销价 × (1 + 增值税率) + 管销后原料
```

**参数说明**：
- 增值税率：默认13%（0.13）
- 管销后原料：原料成本中需要在管销后计算的部分

**外销价计算**：
```
外销价 = 管销价 ÷ 汇率 + 管销后原料
保险价 = 外销价 × (1 + 保险率)
```

**参数说明**：
- 汇率：默认7.2（人民币兑美元）
- 保险率：默认0.3%（0.003）
- 保险价：CIF条款下需要计算保险费

**计算示例（内销）**：
- 管销价：13.125元
- 增值税率：13%
- 管销后原料：2元
- 计算：13.125 × 1.13 + 2 = 16.831元

**计算示例（外销）**：
- 管销价：13.125元
- 汇率：7.2
- 管销后原料：2元
- 保险率：0.3%
- 外销价计算：13.125 ÷ 7.2 + 2 = 3.823元
- 保险价计算：3.823 × 1.003 = 3.834元

#### 3.4.7 自定义费用（管销后费用）

自定义费用允许添加额外的费用项，在管销价之后计算。

**费用项结构**：
- 费用名称：如"特殊税费"、"附加费"
- 费率：百分比（如5%表示0.05）

**计算方式**：
```
最终价格 = 管销价 × (1 + 费率1) × (1 + 费率2) × ... + 管销后原料
```

**累乘计算说明**：
- 多个费用项采用累乘方式
- 例如：费率5%和费率3%，计算为 ×1.05×1.03

**界面展示**：
- 在成本预览面板中显示所有自定义费用项
- 每项显示名称、费率、计算后的金额
- 支持添加、删除、修改费用项

#### 3.4.8 利润区间

利润区间用于展示不同利润率下的建议售价。

**默认档位**：

| 档位 | 利润率 | 用途 |
|------|--------|------|
| 档位1 | 5% | 薄利多销 |
| 档位2 | 10% | 正常利润 |
| 档位3 | 25% | 较高利润 |
| 档位4 | 50% | 高利润 |

**计算公式**：
```
利润价格 = 基础价 ÷ (1 - 利润率)
```

**基础价定义**：
- 内销：管销价
- 外销：管销价 ÷ 汇率

**计算示例**：
- 基础价：13.125元
- 利润率10%：13.125 ÷ (1 - 0.1) = 14.583元
- 利润率25%：13.125 ÷ (1 - 0.25) = 17.5元

**自定义档位**：
- 点击"添加档位"按钮
- 输入自定义利润率（1%-99%）
- 系统自动计算并显示价格
- 支持删除自定义档位（系统默认档位不可删除）

**界面展示**：
- 在成本预览面板底部显示
- 每个档位一行，显示利润率和建议价格
- 价格按从低到高排序

**实时更新机制**：
- 原料、工序、包材明细修改后自动触发计算
- 计算延迟300ms（防抖处理）
- 计算过程中显示loading动画

**成本构成展示**：
- 饼图：展示原料、工序、包材、运费的成本占比
- 条形图：展示各成本项的金额对比
- 图表默认折叠，点击展开

**计算公式汇总**：

| 项目 | 公式 |
|------|------|
| 原料小计(multiply) | (用量 × 单价) ÷ 系数 |
| 原料小计(divide) | (单价 ÷ 用量) ÷ 系数 |
| 工序小计 | 单价 × 数量 × 工价系数 |
| 包材小计 | (单价 ÷ 用量) ÷ 系数 |
| 基础成本 | 原料总价 + 工序总价 + 包材总价 + 运费 |
| 管销价 | (基础成本 + 运费) ÷ (1 - 管销率) |
| 内销价 | 管销价 × (1 + 增值税率) + 管销后原料 |
| 外销价 | 管销价 ÷ 汇率 + 管销后原料 |
| 保险价 | 外销价 × (1 + 保险率) |
| 利润价格 | 基础价 ÷ (1 - 利润率) |

## 六、业务规则

### 6.1 成本计算公式

```
基础成本 = 原料总价 + (工序总价 × 工价系数) + 包材总价 + 运费
管销价 = 基础成本 ÷ (1 - 管销率)
内销价 = 管销价 × (1 + 增值税率) + 管销后原料
外销价 = 管销价 ÷ 汇率 + 管销后原料
保险价 = 外销价 × (1 + 保险率)
```

### 6.2 状态流转

```
draft(草稿) → submitted(已提交) → approved(审核通过)
                              ↘ rejected(已退回) → resubmitted(重新提交) → submitted
```

### 6.3 权限规则

| 操作 | 权限要求 | 数据范围 |
|------|----------|----------|
| 查看 | cost:view | 所有 |
| 创建 | cost:create | - |
| 编辑 | cost:edit | 自己的草稿/已退回，或管理员 |
| 删除 | cost:delete | 自己的草稿，或管理员 |
| 复制 | cost:create | - |
| 设为标准成本 | cost:manage | 已审核状态 |
