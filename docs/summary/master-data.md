# 基础数据模块

## 一、模块概述

### 1.1 定位和作用

基础数据模块是成本分析管理系统的基础支撑模块，负责管理产品成本核算所需的基础数据，包括原料、包材、工序、型号、客户和法规等。这些数据是创建成本分析的必要前提，数据的准确性和完整性直接影响成本计算的准确性。

### 1.2 功能范围

- 原料管理
- 包材管理
- 工序管理
- 型号管理
- 客户管理
- 法规管理

### 1.3 与其他模块的关联

| 关联模块 | 关联关系 | 说明 |
|----------|----------|------|
| 成本分析 | 数据依赖 | 成本分析创建需要选择型号、客户等基础数据 |
| 成本分析 | 计算依赖 | 成本计算需要原料、包材、工序的价格数据 |
| 系统配置 | 参数依赖 | 原料类别等配置数据 |

## 二、原料管理

### 2.1 页面定位

原料管理页面用于维护产品生产所需的原材料信息，包括原料名称、品号、单价、供应商等。支持按产品类别（口罩类/半面罩类）分别管理。

### 2.2 初始化逻辑

1. 从路由参数获取原料类型（half_mask/general）
2. 加载分类结构（子分类列表）
3. 加载原料列表数据
4. 加载型号列表（用于绑定型号）

### 2.3 功能详细描述

#### 2.3.1 视图切换

- **半面罩类视图**：显示半面罩特有字段（供应商、绑定型号、包装方式等）

- **全部视图**：显示所有原料，不区分子分类

#### 2.3.2 搜索功能

## 三、包装配置管理

### 3.1 页面定位

包装配置管理页面用于维护产品的包装层级配置，定义产品从最小单位到运输外箱的包装结构。包装配置是成本计算中包材费用和运费计算的基础，每个型号可以有多个包装配置，但创建成本分析时只能选择其中一个。

### 3.2 四种包装类型

系统支持四种标准包装类型，每种类型对应不同的包装层级结构：

#### 3.2.1 标准彩盒（3层包装）

**层级结构**：片/袋 → 袋/盒 → 盒/箱

**适用场景**：常规零售包装，产品先装入小袋，小袋装入彩盒，彩盒装入外箱

**层级定义**：
- 第一层：每袋片数（如10片/袋）
- 第二层：每盒袋数（如10袋/盒）
- 第三层：每箱盒数（如50盒/箱）

**每箱总数计算**：layer1 × layer2 × layer3（如10×10×50=5000片/箱）

#### 3.2.2 无彩盒（2层包装）

**层级结构**：片/袋 → 袋/箱

**适用场景**：简装产品，省去彩盒环节，小袋直接装入外箱

**层级定义**：
- 第一层：每袋片数（如50片/袋）
- 第二层：每箱袋数（如100袋/箱）
- 第三层：无（固定为1或不显示）

**每箱总数计算**：layer1 × layer2（如50×100=5000片/箱）

#### 3.2.3 泡壳直装（2层包装）

**层级结构**：片/泡壳 → 泡壳/箱

**适用场景**：泡壳包装产品，单片或少量产品装入泡壳，泡壳直接入箱

**层级定义**：
- 第一层：每泡壳片数（如1片/泡壳）
- 第二层：每箱泡壳数（如200个/箱）
- 第三层：无（固定为1或不显示）

**每箱总数计算**：layer1 × layer2（如1×200=200片/箱）

#### 3.2.4 泡壳袋装（3层包装）

**层级结构**：片/袋 → 袋/泡壳 → 泡壳/箱

**适用场景**：袋装后再加泡壳保护，适合需要额外防护的产品

**层级定义**：
- 第一层：每袋片数（如5片/袋）
- 第二层：每泡壳袋数（如4袋/泡壳）
- 第三层：每箱泡壳数（如50个/箱）

**每箱总数计算**：layer1 × layer2 × layer3（如5×4×50=1000片/箱）

### 3.3 包装层级计算逻辑

#### 3.3.1 层级数量输入

**输入控件**：
- 每个层级使用数字输入框
- 最小值为1，最大值为9999
- 支持键盘上下箭头微调

**联动计算**：
- 任一层级数值变化时，实时重新计算每箱总数
- 每箱总数显示在层级输入框下方，格式为"每箱共 XXXX 片"

#### 3.3.2 格式化显示

**包装方式描述**：根据层级自动生成人可读格式

| 包装类型 | 层级值 | 显示格式 |
|---------|--------|---------|
| 标准彩盒 | 10, 10, 50 | "10片/袋, 10袋/盒, 50盒/箱" |
| 无彩盒 | 50, 100, - | "50片/袋, 100袋/箱" |
| 泡壳直装 | 1, 200, - | "1片/泡壳, 200泡壳/箱" |
| 泡壳袋装 | 5, 4, 50 | "5片/袋, 4袋/泡壳, 50泡壳/箱" |

**单位自适应**：
- 第一层单位根据产品类型自动确定（片/个/套）
- 第二层和第三层单位根据包装类型确定（袋/盒/泡壳/箱）

### 3.4 体积计算（CBM）

#### 3.4.1 外箱尺寸输入

**输入字段**：
- 外箱长（cm）：数字输入，保留1位小数
- 外箱宽（cm）：数字输入，保留1位小数
- 外箱高（cm）：数字输入，保留1位小数

**输入布局**：三个输入框横向排列，中间用"×"符号连接，格式为"长 × 宽 × 高"

#### 3.4.2 CBM计算

**计算公式**：CBM = (长 × 宽 × 高) ÷ 1000000

**计算时机**：
- 任一尺寸值变化时实时计算
- 结果显示在尺寸输入框下方

**显示格式**：
- 保留4位小数（如0.0456 CBM）
- 单位标注"立方米"或"CBM"

#### 3.4.3 运费计算关联

CBM用于外销运费计算：
- 运费总价 = CBM × 每CBM运费单价 × 箱数
- 运费成本 = 运费总价 ÷ 订单数量

### 3.5 关联工序和包材

#### 3.5.1 工序关联

**关联方式**：
- 每个包装配置可关联多道工序
- 工序从工序库中选择
- 支持设置每道工序的数量

**工序明细表格**：

| 列名 | 说明 | 编辑方式 |
|------|------|---------|
| 工序名称 | 选择工序后自动填充 | 下拉选择 |
| 工序单价 | 选择工序后自动填充 | 只读 |
| 数量 | 该工序的执行次数 | 数字输入 |
| 小计 | 单价 × 数量 | 自动计算 |
| 操作 | 删除按钮 | 图标按钮 |

**添加工序**：
- 点击"添加工序"按钮
- 弹出工序选择对话框
- 选择工序后自动添加到表格

#### 3.5.2 包材关联

**关联方式**：
- 每个包装配置可关联多个包材
- 包材从包材库中选择
- 支持设置每个包材的用量

**包材明细表格**：

| 列名 | 说明 | 编辑方式 |
|------|------|---------|
| 包材名称 | 选择包材后自动填充 | 下拉选择 |
| 品号 | 选择包材后自动填充 | 只读 |
| 单价 | 选择包材后自动填充 | 只读 |
| 用量 | 每个产品使用的数量 | 数字输入 |
| 单位 | 选择包材后自动填充 | 只读 |
| 小计 | 单价 × 用量 | 自动计算 |
| 操作 | 删除按钮 | 图标按钮 |

**添加包材**：
- 点击"添加包材"按钮
- 弹出包材选择对话框
- 选择包材后自动添加到表格

#### 3.5.3 成本分析自动带入

创建成本分析时：
1. 选择型号后，加载该型号的所有启用状态包装配置
2. 选择包装配置后，自动带入关联的工序和包材明细
3. 带入的明细可编辑（修改数量、删除、添加新项）
4. 计算时自动汇总工序总价和包材总价

- 编辑弹窗中的"以此为基础新建"按钮

**复制流程**：
1. 点击复制按钮
2. 弹出确认对话框，显示源配置信息
3. 确认后打开新增弹窗
4. 弹窗中预填充源配置的所有数据
5. 配置名称自动添加"-复制"后缀
6. 用户可修改后保存为新配置

**复制内容**：
- 型号、生产工厂、包装类型
- 包装层级数量
- 外箱尺寸
- 工序明细（完整复制）
- 包材明细（完整复制）

| 取消 | 次要按钮 | 弹窗底部 | - | 关闭弹窗 |

### 3.8 业务规则

#### 3.8.1 唯一性约束

- 同一型号下，配置名称不能重复
- 违反时提示："该型号下已存在同名配置"

#### 3.8.2 删除限制

- 已被成本分析引用的包装配置不能删除
- 删除时检查引用关系，如有引用提示："该配置已被成本分析引用，无法删除"

#### 3.8.3 状态控制

- 禁用的包装配置在创建成本分析时不可选择
- 状态变更实时生效，不影响已创建的成本分析

#### 3.8.4 数据完整性

- 包装类型变更时，清空层级数量（避免数据混乱）
- 型号变更时，清空工序和包材明细（不同型号的可用工序/包材不同）
